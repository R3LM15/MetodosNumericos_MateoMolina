  1. Sea () = $ -x^3 -cos(x) $ y $_0 = 1$. Use el mtodo de Newton y de la Secante para encontrar 2. Se podra usar $_0$ = 0?

Mtodo de Newton

$$ x_n = x_{n-1} - \frac{f(x_{n-1})}{f'(x_{n-1})} , n \ge 1$$

  • Dado $p_0$ = -1

(x) = $ -x^3 -cos(x) $

'(x) = $ -3x^2 + sen(x) $

Iteracin 1: $$ x_1 = -1 - \frac{0.45969}{-3.84147} = -0.88033$$

(-1) = $ -(-1)^3 -cos(-1) = 0.45969$

'(-1) = $ -3(-1)^2 + sen(-1) = -3.84147$

Iteracin 2: $$ x_2 = -0.88033 - \frac{0.04534}{-3.09589} = -0.86568$$

(-0.88033) = $ -(-0.88033)^3 -cos(-0.88033) = 0.04534$

'(-0.88033) = $ -3(-0.88033)^2 + sen(-0.88033) = -3.09589$

Por lo tanto sabemos que $p_2 = -0.86568$

  • Dado $p_0$ = 0

(0) = $ -(0)^3 -cos(0) = -1 $

'(0) = $ -3(0)^2 + sen(0) = 0 $

El mtodo de Newton falla, porque dividiramos por cero.

Mtodo de la Secante

  • Dado $p_0$ = -1 y eleguido $p_1$ = -1.5

$$ x_n = x_{n-1} - f(x_{n-1}) * \frac{x_{n-1} - x_{n-2}}{f(x_{n-1})-f(x_{n-2})} $$

Clculos:

(-1.5) = $ -(-1.5)^3 -cos(-1.5) = 3.30426 $

(-1) = $ -(-1)^3 -cos(-1) = 0.45969 $

$$ x_2 = -1.5 - 3.30426 * \frac{-1.5 - (-1)}{3.30426-0.45969} = -0.91919 $$

Por lo tanto sabemos que $p_2 = -0.91919$

  • Dado $p_0$ = 0 y eleguido $p_1$ = 1

(0) = $ -(0)^3 -cos(0) = -1 $

(1) = $ -(1)^3 -cos(1) = -1.54030 $

$$ x_2 = 1 - (-1.5403) * \frac{1 - 0}{-1.5403-(-1)} = -1.85082$$

import numpy as np
## metodo de secante (Dos puntos)
def secante(f, x0, x1, tol=1e-4, max_iter=100):
    for i in range(max_iter):
        fx0 = f(x0)
        fx1 = f(x1)

        if fx1 - fx0 == 0:
            print("Divisin por cero en iteracin.")

        x2 = x1 - fx1 * (x1 - x0) / (fx1 - fx0)

        if abs(x2 - x1) < tol:
            return x2  # raz aproximada

        x0, x1 = x1, x2

    print("No converge dentro del nmero mximo de iteraciones.")

# Algoritmo de biseccin
def bisection_method(f, a, b, tol=1e-5, max_iter=100):
    # Verificar que los signos de f(a) y f(b) sean opuestos
    if f(a) * f(b) >= 0:
        print("No se puede aplicar el mtodo de biseccin. Los signos de f(a) y f(b) deben ser opuestos.")
        return None
    
    iter_count = 0
    while (b - a) / 2.0 > tol:
        # Punto medio
        c = (a + b) / 2.0
        # Verificar si el punto medio es una raz
        if f(c) == 0:
            return c
        # Actualizar el intervalo
        elif f(c) * f(a) < 0:
            b = c
        else:
            a = c
        
        iter_count += 1
        if iter_count > max_iter:
            print("Se alcanz el nmero mximo de iteraciones.")
            return None
    
    return (a + b) / 2.0
